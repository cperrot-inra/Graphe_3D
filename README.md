# Graphe

### Visualisation sous forme de graphique en 3D de mesures issues de cartographies saptiales 2D

Cet outil permet d'afficher en quelques clics les graphiques en 3D des données obtenues à partir d'une caractériation de paramètres physiques tels que le rayonnement, la température, etc.

## Prérequis
Windows 7+

## Installation
Exécuter **/app/Graphe 3D/Install/Volume/setup.exe**

## Tests
Fichiers de tests disponibles dans **/Exemples donnees**
